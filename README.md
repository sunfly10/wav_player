# wav_player

### 1. 서론

본 프로젝트는 Arduino Mega 환경에서 SD 카드에 저장된 WAV 파일을 실시간으로 디코딩하여 출력하는 임베디드 플레이어 시스템입니다. 

하드웨어의 자원 한계를 극복하기 위해 Timer/Counter 레지스터 직접 제어와 이중 버퍼링 기법을 적용하여 끊김 없는 고음질 음원 재생을 목표로 제작되었습니다

### 2. 주요 기능

•고성능 음원 재생 및 멀티 포맷 지원

포맷 호환성: Stereo/Mono, 16-bit/8-bit, 44.1kHz/22.05kHz 등 다양한 규격의 WAV 파일 재생이 가능합니다.

이중 버퍼링(Double Buffering): 3,000바이트 크기의 버퍼 2개를 운용하여, 한 버퍼가 재생되는 동안 다른 버퍼를 채우는 방식으로 실시간성을 확보했습니다.

PWM 음원 출력: Timer/Counter 2와 4를 활용하여 생성한 8-bit PWM 신호를 통해 스테레오 사운드를 출력합니다.

•지능형 사용자 인터페이스 

네비게이션: 로터리 엔코더(INT3)를 이용해 곡을 탐색하고, 푸시 버튼(INT1)으로 재생/정지 및 폴더 진입을 제어합니다.

계층형 파일 시스템: SdFat 라이브러리를 통해 일반적인 파일 형식을 지원하며, 상위 폴더 이동([<-]) 기능을 지원합니다.

실시간 정보 출력: LCD 첫 줄에 곡명(최대 16자), 둘째 줄에 재생 상태 및 진행률(%)을 표시합니다.

시간 이동 기능: 가변저항(VR1)으로 이동 단위를 설정(1~7초)하고 스위치(SW1, SW2)를 통해 앞뒤로 구간 점프가 가능합니다.

### 3. 기술적 세부 사항

•하드웨어 레지스터 직접 제어

LCD 드라이버: LiquidCrystal 라이브러리에 의존하지 않고, 레지스터 쓰기 방식을 직접 설계하여 처리 속도를 높였습니다.

샘플링 동기화: Timer/Counter 1 인터럽트를 활용하여 각 음원의 샘플링 레이트에 맞춰 정밀하게 PWM 데이터를 업데이트합니다.

•문제 해결 및 최적화 

음질 저하 및 노이즈 해결: 16-bit 음원 재생 시 발생하는 왜곡을 방지하기 위해 소프트웨어적 게인 조절(Right Shift) 기법을 적용했습니다.

전력 관리: 고출력 구간에서 발생하는 LCD 깜빡임 현상을 분석하여 전력 소모를 최적화하고 시스템 리셋을 예방했습니다.

메모리 최적화: 아두이노 메가의 SRAM 용량을 고려하여 파일 이름 길이와 목록 개수를 제한하여 시스템 안정성을 높였습니다.
